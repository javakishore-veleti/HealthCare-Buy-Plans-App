# Developer Local Laptop Setup Notes

## Installation
1. Python 3.11 or 3.12
2. Git command line 
	- Windows -> https://git-scm.com/install/windows -> Standalone Installer (Git for Windows/x64 Setup)
	- Macbook -> Homebrew -> run the command -> brew install git
3. PyCharm Free Edition 
	- Windows -> https://www.jetbrains.com/pycharm/download/?section=windows
	- Macbook -> https://www.jetbrains.com/pycharm/download/?section=mac
4. Visual Studio code 
	- Windows -> https://code.visualstudio.com/docs/?dv=win32arm64user
	- Macbook -> https://code.visualstudio.com/
5. Notepad++ if possible only (only for Windows users not for Macbook users)
6. sublimetext if possible only  https://www.sublimetext.com/ (both for Windows and Macbook Users)
7. Mysql database 
	- Download MySQL Workbench -> https://dev.mysql.com/downloads/workbench/
	- Select Mac or Windows based on your operating system
8. postgres database (optioinal)
	- https://www.enterprisedb.com/downloads/postgres-postgresql-downloads
	- Install 15.15 version and dont install any other higher versions
	- If you already have Postgres installed then skip this step
	- If you already have Postgres higher version than 15.15 it is fine
9. Docker desktop 
	Windows -> https://docs.docker.com/desktop/setup/install/windows-install/
		Docker Desktop for Windows - x86_64
	Macbook -> https://docs.docker.com/desktop/setup/install/mac-install/
		Docker Desktop for Mac with Apple silicon
10. For UI Developent Install angular related software
	10.1 -> Node.js installation -> https://nodejs.org/en/download
		Windows -> Windows Installer (.msi)
		Macbook -> macOS Installer (.pkg)
